# 可莉档案袋改造设计

日期：2026-04-16

## 目标

把现有的科幻 HUD 作品集模板改造成“西风骑士团机密档案袋”，主题角色为可莉。

## 设计结论

- 保留模板最强的部分：加载动画、六列导航、左侧控制面板、详情页转场。
- 不做纯官方角色页复刻，而是做“官方资料结构 + 骑士团卷宗包装”。
- 整体内容为混合向：前半是角色资料，后半是图像与附录材料。

## 栏目映射

- `works` -> 基础档案
- `experience` -> 作战天赋
- `blog` -> 故事记录
- `life` -> 图像档案
- `contact` -> 附录备注
- `about` -> 终页摘要

说明：

- 命之座不单独占一列，放入“基础档案”区作为第二个 tab。
- 这样既保留 6 列结构，又能让命座内容拥有独立入口感。

## 文案口径

- 主体内容尽量稳、准、接近官方角色资料结构。
- 边角提示、系统 HUD、附录标题使用卷宗包装语气。
- 比例控制为“80% 档案感，20% 可莉气息”。

## 首批改造范围

- 全局站名、SEO、加载文案、HUD 文案
- 首页六列名称与左侧导航名称
- `data/*` 内容源整体替换为可莉主题占位数据
- `content/blog/*` 改为故事记录示例
- `friends` 页面改为关联角色页

## 暂不处理

- 真实角色素材替换
- 深度样式重绘与颜色系统重构
- 更复杂的新交互或新数据结构
